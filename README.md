# Teacher AI – AI-Powered Learning Platform

Teacher AI is an intelligent platform that transforms teachers’ notes, slides, or lesson materials into **personalized quizzes, homework, and study reviews**. It adapts to student performance, helping students focus on their strengths and weaknesses while saving teachers time.

## Features

- **Automatic Exam Generation:** Convert uploaded notes or PDFs into structured practice exams.  
- **Personalized Quizzes:** AI generates topic-specific questions and explanations based on lesson content.  
- **Homework & Study Reviews:** Provides additional exercises and summaries for reinforcement.  
- **Adaptive Learning:** Tailors future quizzes based on student performance.  
- **Interactive Web Platform:** Responsive interface for seamless teacher and student experience.  

## How It Works

1. Upload notes or slides (PDF).  
2. AI processes the content using **Natural Language Processing (NLP)** and **Gemini AI**.  
3. System generates multiple question types organized by topic.  
4. Students take quizzes, review summaries, and track progress.
